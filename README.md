# SparkCommerceProject
Spark电商数据分析平台，技术栈：Spark+kafka+Hive，以完成需求为导向，熟悉SparkCore、SparkSql和SparkStreaming的Api和开发技巧
适合刚学完Spark和Kafka的初学者练手；

## 项目结构
> ├─commons ：项目配置工具类、常量接口和工具类等

> ├─mock ：负责模拟生成离线和实时数据

> ├─adversStat ：广告点击流量实时统计

> ├─areaStat ：各区域热门商品统计

> ├─pageStat ：页面单跳转换率分析

> ├─session ：用户访问session分析
